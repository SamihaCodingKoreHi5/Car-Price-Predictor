# 🚗 Car Price Predictor

A beginner-friendly Machine Learning project that predicts used car prices using real-world vehicle data from the CarDekho dataset.

## 📌 Project Overview

This project uses Python, Pandas, and Scikit-Learn to analyze car market data and build a Linear Regression model for predicting car prices.

The model learns from historical vehicle information and estimates the selling price of a car based on its features.

---

## ✨ Features

- Load and analyze a real-world car dataset
- Explore car market statistics
- Identify top car brands
- Analyze fuel type distribution
- Find the most expensive and cheapest cars
- Train a Linear Regression model
- Predict car prices based on vehicle information

---
